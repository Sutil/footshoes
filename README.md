## English

ReactJS project to study of:

* Layout;
* E-commerce concepts;
* Redux;
* Redux Saga;
* Routes;
* Backend mock with json-server;
* Reactotron;
* eslint;
* Others;

### To run

```
yarn global add json-server
json-server server.json -p 3333
```

Open other terminal and run:

```
yarn start
```
